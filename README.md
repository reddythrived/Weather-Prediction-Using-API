# 🌦 Weather Prediction App

A production-ready web application that provides real-time weather insights and short-term forecasts using external API integration.

---

## 🚀 Live Application

https://web-production-398c6.up.railway.app/

---

## ⚡ Key Features

* Real-time weather data by city
* 3-day forecast with structured insights
* Robust error handling for invalid inputs
* Clean, responsive user interface

---

## 🛠 Tech Stack

* **Backend:** Flask (Python)
* **Frontend:** HTML, CSS
* **API:** OpenWeather
* **Deployment:** Railway (Gunicorn)

---

## 🔐 Security & Best Practices

* API key managed via **environment variables**
* Sensitive data excluded using `.gitignore`
* Production-ready server configuration

---

## ⚙️ Local Setup

```bash
git clone https://github.com/reddythrived/Weather-Prediction-Using-API.git
cd Weather-Prediction-Using-API
pip install -r requirements.txt
```

Create `.env`:

```bash
API_KEY=your_openweather_api_key
```

Run:

```bash
python predict.py
```
